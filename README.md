# ResQNet - Mesh-Based Emergency Communication App

ResQNet is an Android app that helps users communicate during emergencies without internet by using nearby devices and offline messaging.

## Features
- **Offline Chat:** Communicate with nearby users using Nearby Service.  
- **Map Integration:** View and locate nearby users on Google Maps.  
- **SOS Alerts:** Send emergency alerts from the app to the admin dashboard.  
- **User Management:** Admin can manage users and emergencies from the website.  
- **Navigation & Permissions:** Smooth onboarding with location and nearby device permissions.

## Tech Stack
- **Frontend (App):** Android (Java)  
- **Backend:** React
- **Database:** Firestore and RoomDB  
- **Mapping:** Google Maps API  
- **Testing & Documentation:** Postman
